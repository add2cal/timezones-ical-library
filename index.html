iCal Timezone information for Javascript

This tool converts the most recent IANA (formerly Olson) timezone database files into VTIMEZONE blocks, which are compatible with the iCalendar specification (RFC2445).
In an easy to use way for every Javascript project; may it be VanillaJS, React, Angular, Vue, Svelte, or based on any other framework or library.
